Lizenzvertrag zwischen dem
\begin{center}
Konrad-Zuse-Institut für Informationstechnik\\
Takustraße 7, 14195 Berlin-Dahlen\\
\end{center}
im folgenden ZIB genannt und der
Fraunhofer Gesellschaft für angewandte Forschung e.V. für ihr
\begin{center}
Fraunhofer-Institute for Algorithms
and Scientific Computing\\
Schloss Birlinghoven, 53754 Sankt Augustin
\end{center}
in folgenden Lizenznehmer genannt, 
über das Softwarepacket SoPlex Version 1.2.1, im Quellkode
einschließlich der dazugehörigen Dokumentation, 
im folgenden Software gennant:
}

\begin{enumerate}

\item Das ZIB erteilt dem Lizenznehmer eine zeitlich, räumlich und
   inhaltlich unbeschränkte, nicht ausschließliche Lizenz zur
   Benutzung und Verwertung des Softwarepacketes SoPlex.

   Der Lizenznehmer ist insbesondere berechtigt, die Software für
   eigenen Zwecke zu nutzen, weiter zu entwickeln, in andere
   Softwaresysteme zu integrieren und die Software zu diesem Zweck zu
   vervielfältigen, zu bearbeiten, zu verändern, vorzuführen und zu
   übermitteln.

   Insbesondere darf die Software als sogennante Bibliothek in
   binärer Form an Softwareprodukte gebunden und eingesetzt werden.
   In dieser Form unterliegt die Weitergabe der Software keinen
   Beschränkungen. 

   Eine Weitergabe der Software im Quellkode an Dritte, 
   sofern diese an Projektpartner des Lizenznehmers im
   Rahmen eines Forschungsprojekts zur ausschließlichen Verwendung im
   und während des Projekts erfolgt, ist zulässig, aber dem ZIB
   anzuzeigen. 

   In allen übrigen Fällen einer Weitergabe bzw. (Unter-)Lizenzierung
   der Software als Quellcode an Dritte Bedarf diese der vorherigen
   schriftlichen Zustimmung des ZIB.

\item Das ZIB gewährleistet, die Software selbst entwickelt zu haben und daß
     die nach diesem Vertrag eingeräumten Nutzungsrechte des Lizenznehmers
     keine Rechte Dritter verletzen.

\item Diese Lizenz gilt auch für alle nachfolgenden Versionen der
   Software die vom ZIB frei gegeben werden.

\item Bei Vertragsabschluß gilt die Software als vollständig übergeben.

\item Das Eigentumsrecht an der Software verbleibt beim ZIB.

\item Das Urheberrecht für die Software verbleibt beim ZIB. 

\item Das ZIB überläßt dem Lizenznehmer die Software \glqq so, wie sie
   ist\grqq und steht nicht dafür ein, daß die Software für die
   Zwecke des Lizenznehmers brauchbar ist.

\item Gewährleistungsansprüche an das ZIB sind ausgeschlossen.

\item Das ZIB übernimmt keine Wartungsverpflichtungen.

\item Das ZIB übernimmt keine Haftung für Schäden jeglicher Art, die
   sich aus der Verwendung der Software ergeben. 

\item Lizenznehmer zahlt für die Lizenz nach Rechnungstellung durch
   das ZIB eine einmalige Lizenzgebühr in Höhe von Euro 18.000
   zuzüglich der gesetzlichen Mehrwertsteuer.
 
\item Als Gerichtsstand ist Berlin vereinbart.

\end{enumerate}
